# Projet Hôtel — Mini-site Web

## Présentation
Ce projet est un mini-site web réalisé dans le cadre du module
**Développement Web (L2 – S4)**.

Il présente le projet GLP **Gestion d’un hôtel**, sous la forme
d’un site de communication professionnelle.

Le site décrit le concept du projet, son cahier des charges,
l’équipe, l’état d’avancement et les coordonnées de contact.

## Objectifs pédagogiques
- Utilisation correcte du HTML5 sémantique
- Mise en œuvre du CSS (externe, interne et local)
- Création d’un site web multi-pages cohérent
- Réalisation d’un menu de navigation horizontal
- Respect des bonnes pratiques d’ergonomie et d’accessibilité

---

## Structure du site

- `index.html` : page d’accueil et présentation générale
- `projet.html` : description détaillée du projet
- `specifications-projet.html` : synthèse du cahier des charges
- `equipe.html` : présentation de l’équipe
- `realisations.html` : état d’avancement du projet
- `contact.html` : coordonnées de contact
- `style.css` : feuille de style externe commune
- `images/` : ressources graphiques (logo, illustrations)

---

## Technologies utilisées
- HTML5
- CSS3

Aucun framework ou bibliothèque externe n’a été utilisé.

---

## Remarque
Ce site est réalisé dans un cadre universitaire.
Les fonctionnalités décrites correspondent à des objectifs de conception
et pourront évoluer lors du développement du projet.
